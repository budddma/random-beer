#### Инструкция по запуску бэкенда
1. `cd backend`
2. `virtualenv env`
3. `source env/bin/activate`
4. `pip install -r requirements.txt`
5. `./manage.py runserver`

#### Инструкция по запуску фронтенда
1. `cd frontend`
2. `npm install`
3. `npm start`
